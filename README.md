# Hello Django 应用

这是一个基础的Django Hello应用，展示了Django框架的基本功能。

## 功能特点

- 显示"你好，世界！"的首页
- 支持通过URL参数进行个性化问候
- 简洁的页面设计

## 安装与运行

1. 克隆仓库到本地
```
git clone https://github.com/你的用户名/hello-django-app.git
cd hello-django-app
```

2. 运行开发服务器
```
python manage.py runserver
```

3. 在浏览器中访问
```
http://127.0.0.1:8000/
```

## 项目结构

- `hello_project/` - Django项目配置
- `hello/` - Hello应用代码
- `templates/` - HTML模板文件
